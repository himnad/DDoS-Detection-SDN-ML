# Contributing Guidelines

Thank you for your interest in contributing to the **DDoS Detection and Mitigation using Machine Learning (SDN)** project. Contributions are welcome and help improve the overall quality and usability of this project.

---

## Ways to Contribute

You can contribute in the following ways:

- **Bug Reports**  
  Identify and report issues with clear steps to reproduce.

- **Feature Requests**  
  Suggest new features or improvements by creating an issue.

- **Documentation Improvements**  
  Enhance clarity, fix errors, or add missing information.

- **Code Contributions**  
  Improve existing functionality or add new modules.

---
